# Teaching Calendar — AI Agent Guide

> 南昌中小学教学进度日历，按年级/科目/学期/周次组织，供 AI Agent 和教师查询使用。
> **任何修改后必须同步更新 Teaching-System/AGENTS.md 的变更日志。**

## 系统定位

本仓库是教学备课系统的**数据源之一**，为 AI 备课助手提供学校教学进度参考。
配套仓库见 [Teaching-System/AGENTS.md](https://github.com/1686756626/Teaching-System/blob/main/AGENTS.md)。

## Quick Start

**一句话说明这个仓库是什么：**
南昌市七年级到高三的语文、历史、政治（道法）学科教学进度参考，按周次划分，覆盖春/秋两学期。

**AI 如何使用：**
1. 读 `校历总览.md` — 获取学期日期、假期、考试安排
2. 读 `src/{学段}/{年级}/{科目}/{学期}.md` — 获取某年级某科目的详细教学进度

> ⚠️ **`src/` 下的 .md 文件是唯一数据源。** `schedule.json` 已不再被网站使用，仅供 AI 备课时可选参考。如果 .md 和 schedule.json 有冲突，以 .md 为准。

## .md 文件格式规范

每个 `src/` 下的 .md 文件遵循统一格式，**修改时必须保持一致**：

```markdown
# {年级}{科目}（{册别}）教学进度 — {备注}

> 教材：...

## {月份}

### 第{起始周}-{结束周}周（{起始日期} - {结束日期}）
- 教学内容（**重点内容用粗体标注**）
- ...
```

### 格式要点

| 元素 | 规范 | 示例 |
|------|------|------|
| 月份标题 | `## X月` 或 `## X-Y月` | `## 3月`、`## 4月` |
| 周次标题 | `### 第N周（M.D - M.D）` 或 `### 第N-M周（M.D - M.D）` | `### 第1-2周（2.13 - 2.26）` |
| 内容行 | 以 `- ` 开头，重点用 `**粗体**` | `- **第一轮复习：古诗文**` |
| 补充内容 | 用 `---` 分隔后附在文件末尾 | 中考重点篇目速查等 |

### 前端解析逻辑

网站通过 `parseSubjectMd()` 解析 .md 文件：
- 从 `### 第N周（M.D - M.D）` 提取周次和日期
- 从 `## X月` 提取月份（用于表格月份分隔行）
- 从 `- ` 列表提取教学内容
- 从 `**文本**` 提取重点（自动填入"教学重点"列）
- `---` 之后的内容不参与周次表格渲染（仅用于详情查看）

## 校历总览.md 格式

```markdown
## 第一学期（秋季）
| 项目 | 时间 |
|------|------|
| 开学 | 9月1日 |
| ...
| 全学期 | 约20周 |

## 第二学期（春季）
| 项目 | 时间 |
|------|------|
| 开学 | 2月13日 |
| ...
| 全学期 | 约20周 |
```

### 前端解析逻辑

网站通过 `parseSchoolCalendarMd()` 解析，自动提取：
- **学期起始日期** → `CONFIG.semester.start`（从"开学"字段解析月/日）
- **总周数** → `CONFIG.semester.totalWeeks`（从"全学期"字段解析数字）
- **当前学期** → 根据当前月份自动判断（2-7月=第二学期）
