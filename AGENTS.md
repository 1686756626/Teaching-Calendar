# Teaching Calendar — AI Agent Guide

> 南昌中小学教学进度日历，按年级/科目/学期/周次组织，供 AI Agent 和教师查询使用。

## 系统定位

本仓库是教学备课系统的**数据源之一**，为 AI 备课助手提供学校教学进度参考。
配套仓库见 [Teaching-System/AGENTS.md](https://github.com/1686756626/Teaching-System/blob/main/AGENTS.md)。

## Quick Start

**一句话说明这个仓库是什么：**
南昌市七年级到高三的语文、历史、政治（道法）学科教学进度参考，按周次划分，覆盖春/秋两学期。

**AI 如何使用：**
1. 读 `schedule.json` — 所有进度数据的结构化版本（优先）
2. 需要某年级科目的详细教学计划 → 读 `src/` 下对应的 `.md` 文件

## 数据结构 (schedule.json)

```json
{
  "meta": { "适用地区": "...", "教材版本": "..." },
  "校历": {
    "第一学期": { "开学": "9月1日", "期中考试": "...", ... },
    "第二学期": { "开学": "2月13日", "期中考试": "...", ... }
  },
  "初中": {
    "七年级": {
      "语文": {
        "上学期": [{ "周次": 1, "日期": "9.1-9.6", "月份": "9月", "内容": "...", "重点": "..." }],
        "下学期": [...]
      },
      "历史": { "上学期": [...], "下学期": [...] },
      "道德与法治": { "上学期": [...], "下学期": [...] }
    }
  },
  "高中": {
    "高一": {
      "语文": { "上学期": [...], "下学期": [...] },
      "思想政治": { "上学期": [...], "下学期": [...] },
      "历史": { "上学期": [...], "下学期": [...] }
    }
  }
}
```

> ⚠️ **注意实际结构是三级嵌套 `学段/年级/科目`，不是 `学段-年级-科目` 拼接的 key。**

查询示例：
- 查七年级语文第10周 → `初中.七年级.语文.上学期[9]`（0-indexed）
- 查某月所有内容 → 遍历筛选 `月份` 字段
- 查九年级下学期（中考冲刺）→ `初中.九年级.语文.下学期`

### 科目名称（重要）

| 学段 | schedule.json 中的 key | 常用简称 |
|------|----------------------|----------|
| 初中 | `道德与法治` | 道法 |
| 高中 | `思想政治` | 政治 |
| 通用 | `语文` | 语文 |
| 通用 | `历史` | 历史 |

## 目录结构

```
├── AGENTS.md              ← 你正在读的文件
├── schedule.json          ← 结构化数据（AI 优先读这个）
├── 校历总览.md             ← 南昌市学期安排
└── src/                   ← 各年级科目详细教学计划（markdown）
    ├── 初中/
    │   ├── 七年级/{语文,历史,道德与法治}/{上学期,下学期}.md
    │   ├── 八年级/{语文,历史,道德与法治}/{上学期,下学期}.md
    │   └── 九年级/{语文,历史,道德与法治}/{上学期,下学期}.md
    └── 高中/
        ├── 高一/{语文,历史,思想政治}/{上学期,下学期}.md
        ├── 高二/{语文,历史,思想政治}/{上学期,下学期}.md
        └── 高三/{语文,历史,思想政治}/{上学期,下学期}.md
```

> `src/` 下的 `.md` 文件内容比 `schedule.json` 更详尽，包含完整的教学安排、复习节奏、重点篇目等。
> 前端教学日历页面的"查看详细教学计划"按钮即读取这些文件。

## 注意事项

- 数据基于统编版教材常见教学节奏，各校可能有1-2周偏差
- 南昌市校历：期中约第10-11周，秋季寒假约1月中旬，春季暑假约6月底-7月初
- 九年级下学期和高三下学期为考试冲刺节奏，非新课
- 最后更新：2025年4月
