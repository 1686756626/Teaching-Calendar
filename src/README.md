## 使用建议

1. **每次上课前花2分钟问学生学校讲到哪了**，对照进度表判断快慢
2. **期中、期末前2-3周**开始冲刺辅导，是家长需求最旺盛的时期
3. **初三下学期3月**是政治历史冲刺的最佳入场时机
4. **高三9月**可以接高考历史的一轮复习辅导
5. 各学校进度可能有1-2周差异，以学生实际情况为准
